= データのソート
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
結果をソートするには、列をクリックし、昇順または降順を指定します。データをソートすることで、レポートに必要な情報に優先順位を付けることができます。

.手順
. 列の上部をクリックして結果をソートし、最も重要な情報がビューの上部に表示されるようにします。
. 保存されていないビューに名前を付けて変更を保存します。
