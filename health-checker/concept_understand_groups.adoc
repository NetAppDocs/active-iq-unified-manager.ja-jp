= グループの概要
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Unified Manager でグループを作成してストレージオブジェクトを管理できます。グループの概念とグループルールを使用してストレージオブジェクトをグループに追加する方法を理解しておくと、環境内のストレージオブジェクトを管理するのに役立ちます。
