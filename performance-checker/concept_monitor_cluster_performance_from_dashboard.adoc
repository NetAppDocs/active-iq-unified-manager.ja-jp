= ダッシュボードからのクラスタパフォーマンスの監視
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Unified Manager のダッシュボードには、 Unified Manager の現在のインスタンスで監視しているすべてのクラスタのパフォーマンスステータスの概要が、いくつかのパネルに分けて表示されます。管理対象クラスタの全体的なパフォーマンスを評価し、特定のイベントをすばやく把握して特定し、解決策を適用することができます。
