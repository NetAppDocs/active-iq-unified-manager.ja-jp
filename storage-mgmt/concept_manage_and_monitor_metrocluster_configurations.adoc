= MetroCluster 構成を管理および監視する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Unified Manager Web UIでは、MetroCluster 構成を監視することで、MetroCluster over FCおよびIP構成の接続に問題がないかを確認できます。接続問題を早期に検出することで、 MetroCluster 構成を効果的に管理できます。
